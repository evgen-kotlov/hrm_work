About QA Tech HR System

System Overview

QA Tech HR Management System is a comprehensive web application designed for managing employee information, documents, and generating reports. This application serves as an excellent platform for practicing test automation with various user roles, form validations, and complex UI interactions.

User Roles & Permissions

  • Admin User (Username: Admin, Password: admin123) - Full access to all features including employee management, document upload, report generation, and system settings
  • Regular User (Username: User, Password: user123) - Limited access with view-only permissions for most features, cannot modify data or access settings

Core Features

  • Employee Management - Add, view, edit, and delete employee records with photo uploads
  • Document Management - Upload, download, and organize company documents
  • Reporting System - Generate various reports (Employee List, Department Summary, Salary Report) in multiple formats
  • Dashboard Analytics - Visual statistics and recent activities overview
  • Search & Filtering - Find employees quickly with real-time search functionality
  • Pagination - Navigate through large datasets efficiently
  • Role-based Access Control - Different permissions for admin and regular users

Input Validation Rules

First Name & Last Name:
  • Only letters, spaces, apostrophes (') and hyphens (-) allowed
  • Maximum 50 characters
  • Required field
Employee ID:
  • Only letters, numbers, hyphens (-) and underscores (_) allowed
  • Maximum 20 characters
  • Optional field
Email Address:
  • Must follow standard email format (user@domain.com)
  • Maximum 100 characters
  • Required field
Position:
  • Letters, numbers, spaces and basic punctuation allowed
  • Maximum 100 characters
  • Optional field
Salary:
  • Must be a number between 0 and 1,000,000
  • Increments of 1000
  • Optional field
Additional Notes:
  • Maximum 500 characters
  • Real-time character counter with color indicators
  • Optional field
File Uploads:
  • Maximum file size: 10MB
  • Photo upload supported for employee profiles
  • Multiple document upload supported

Technical Features

  • Responsive Design - Works on desktop, tablet, and mobile devices
  • Modern UI/UX - Gradient colors, smooth animations, and intuitive navigation
  • Local Storage - All data persists in browser local storage
  • Form Validation - Real-time validation with visual feedback
  • Toast Notifications - User-friendly success and error messages
  • Modal Windows - For detailed views and additional actions
  • Tabbed Interfaces - Organized form sections for better user experience
  • Iframe Integration - External content display with dynamic source changing

Test Automation Opportunities

This application provides numerous scenarios for test automation practice:

  • User authentication and role-based testing
  • Form validation testing with various input scenarios
  • CRUD (Create, Read, Update, Delete) operations testing
  • File upload and download functionality testing
  • Search and pagination testing
  • Modal and popup interaction testing
  • Dynamic content and AJAX operation testing
  • Cross-browser compatibility testing
  • Responsive design testing
