# Автотесты для HR Management System

## Предварительные требования
- Node.js (версия 18 или выше)
- npm или yarn
- Запущенный бэкенд сервер (из папки с проектом выполните `npm run server`)

## Установка
1. Клонируйте репозиторий с тестами.
2. Перейдите в директорию `tests`.
3. Выполните `npm install` для установки зависимостей.

## Запуск тестов
- Запустить все тесты: `npm test`
- Запустить smoke-тесты: `npm run test:smoke`
- Запустить регрессионные тесты: `npm run test:regress`
- Запустить тесты в режиме с UI: `npm run test:headed`
